Easily add expandable "Read More/Read Less" content toggles to your pages. Fully customizable via the settings page.

== Description ==

**ReadMore ReadLess** lets you hide/show any content within your posts and pages with a smooth animated toggle and fully customizable button.

- Use the [readmore]...[/readmore] shortcode to create expandable/collapsible content blocks anywhere.
- Customize button background color, font color, font size, and button text ("Read More"/"Read Less") from your WordPress admin's settings page.
- Includes responsive design, smooth CSS transitions, and accessibility enhancements.
- "Settings" link is available directly from the Plugins page for one-click access.

== Installation ==

1. Upload the `ReadMore-ReadLess` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Use [readmore] ... [/readmore] in posts or pages. Hidden content goes inside the shortcode.

== Frequently Asked Questions ==

= How do I change the button color or text? =
Go to **Settings > Read More** in your WordPress admin. Set background color, font color, button text, and font size as you wish.

= Can I use multiple [readmore] blocks on a single page? =
Yes! Each block works independently.

= Does it work with the block/Gutenberg editor? =
Yes. Use the "Shortcode" block or paste the shortcode directly.
